1. はじめに：AIと一緒に「つくる」時代へ

2026年、プログラミングの世界は大きく変わりました。これまでの「コードを一行ずつ書く」スタイルから、AIのエージェントと一緒に「どんなアプリを作るか」を考えるスタイルへと進化したのです。その中心にあるのが、Google Antigravityです。

Antigravityは、単なる便利ツールではありません。面倒な作業はAIに任せて、あなたは「どんな体験をユーザーに届けたいか」という、一番クリエイティブな部分に集中できる。そんな新しい開発のパートナーです。

このガイドでは、プログラミングが初めての方や、AIを使ってみたい方向けに、Antigravityの魅力をわかりやすく解説します。難しいことは抜きにして、AIと一緒にアプリを作る「Vibe Coding（バイブ・コーディング）」の楽しさを体験してみましょう。

2. Antigravityの仕組みと「3つの作業場」

Antigravityでは、AIエージェントが3つの場所（サーフェス）を行き来して、あなたの代わりに作業をしてくれます。

2.1 3つの作業場（サーフェス）

Editor (エディタ)

役割: コードを書いたり、ファイルを整理したりする場所。

AIができること: 複数のファイルを一度に開いて、全体を見ながら修正してくれます。「ここを直して」と頼むだけでOK。

Terminal (ターミナル)

役割: アプリを動かすための準備や、テストをする場所。

AIができること: エラーが出たら勝手に原因を調べて、必要なソフトをインストールしたりして解決してくれます。

Browser (ブラウザ)

役割: 出来上がったアプリの画面を確認する場所。

AIができること: 実際に画面を操作して、「ボタンがちゃんと動くか」などをチェックしてくれます。

2.2 Agent Manager：頼れる司令塔

Agent Managerは、プロジェクトの司令塔です。あなたはここで「次はこれを作って」「バグを探して」と指示を出すだけ。複数のAIエージェントを同時に働かせることもできます。

2.3 賢い頭脳：Gemini 3

  • Gemini 3 Pro (Deep Think): じっくり考えるのが得意。難しい設計や、原因不明のエラー解決にお勧め。
  • Gemini 3 Flash: 素早くて安い。簡単なコード作成や、ちょっとした修正にピッタリ。
  • その他のAI: ClaudeやGPTなど、好みのAIモデルを選ぶこともできます。

3. Vibe Coding：ノリと雰囲気でつくる

「Vibe Coding（バイブ・コーディング）」は、細かいコードの書き方を気にせず、「こんな感じのアプリがいいな」「もっとカッコよくして」といった「雰囲気（Vibe）」をAIに伝えて作るスタイルです。

おすすめの頼み方（ポモドーロタイマーを作る場合）
"Create a functional Pomodoro timer web app using HTML, CSS, and JavaScript. It should have a 25-minute default timer, start/pause/reset buttons, and a visual progress ring. Use Tailwind CSS for a modern, clean aesthetic. The design should evoke a sense of calm and focus."

3.2 初心者のためのステップ

  1. ステップ1：やりたいことを伝える - 「何を作りたいか」をはっきり伝えましょう。
  2. ステップ2：計画を確認する - AIが「こうやって作ります」と計画を出してくるので、OKを出します。
  3. ステップ3：出来上がりをチェック - 作ってくれたアプリを実際に触ってみて、確認します。
  4. ステップ4：会話しながら直す - 「ここはもっとこうして」と会話するように修正していきます。

4. Agent Skills：AIに道具を持たせる

Agent Skillsは、AIに特定の作業専用の「道具」や「知識」を渡す機能です。これを使うと、AIはもっと賢くなります。

4.3 最初に入れておきたいスキル

Git Commit Formatter

作業の保存（コミット）をするとき、わかりやすいメッセージを自動で書いてくれます。

Systematic Debugging

エラーが起きたとき、プロのように順序立てて原因を探ってくれます。

Documentation Reader

公式の説明書（ドキュメント）を読んで、正しい書き方を調べてくれます。

Deployment Skill

出来上がったアプリを、ボタン一つでインターネット上に公開してくれます。

5. AIの考えを見える化する

AIが勝手に変なことをしないか心配ですか？Antigravityは、AIが何を考えているかちゃんと教えてくれるので安心です。

5.1 安心のための仕組み

  • Implementation Plan (計画書): 作業を始める前に、「これから何をするか」を書いて見せてくれます。
  • Task List (やることリスト): 具体的な作業の手順をリストアップしてくれます。
  • Verification (確認): 「ちゃんと動きましたよ」という証拠（ログやスクショ）を見せてくれます。

6. コストと学習のコツ

AIを使うには少しお金（利用料）がかかります。賢く使って、無駄を減らしましょう。

お得に使うコツ

  • モデルを使い分ける: 簡単な作業は安い「Flash」モデルを使いましょう。
  • 道具は必要な分だけ: 使わないスキルはオフにしておくと、AIが迷いません。
  • こまめにリセット: ひとつの作業が終わったら、会話をリセットするとスッキリします。

8. おわりに

Google Antigravityを使えば、プログラミングの知識がなくても、あなたのアイデアを形にすることができます。大切なのは「何を作りたいか」という情熱だけです。

さあ、怖がらずに始めてみましょう。Antigravityを開いて、最初の一言を打ち込めば、あなたも立派な「クリエイター」です。
